About Animal Classification

Project Overview

This web application uses a deep learning model to classify images of three wildlife species: Elephants, Monkeys, and Wild Boars. The model was trained using convolutional neural networks (CNN) to recognize distinctive features of each animal.

Our system helps in wildlife monitoring and conservation efforts by automatically identifying these animals in images, which can be particularly useful for researchers and conservationists.

How It Works

  1. Upload an image containing an Elephant, Monkey, or Wild Boar
  2. The image is processed and resized to fit our model's requirements
  3. Our deep learning model analyzes the image features and patterns
  4. The system predicts which animal is in the image with a confidence score
  5. Results are displayed along with interesting facts about the identified animal

Technical Details

  • Framework: Flask (Python web framework)
  • Deep Learning: TensorFlow/Keras
  • Model Architecture: Convolutional Neural Network (CNN) with transfer learning
  • Image Processing: 224x224 pixel normalization
  • Frontend: HTML, CSS, Bootstrap 5
  • Accuracy: ~95% on validation set

The model was trained on thousands of images to ensure accurate classification across different angles, lighting conditions, and backgrounds.
